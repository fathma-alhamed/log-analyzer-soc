# Log Analyzer (SOC Project)

## Description
A Python-based log analyzer that detects suspicious login activity using regex parsing.

## Features
- Detects brute force attempts
- Tracks IP activity
- Flags suspicious behavior

## Tech Used
- Python
- Regex
- Collections module
